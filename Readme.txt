Stream Alive checks whether or not a stream is alive using http://twitch.tv api along with checking the image for a streamer being live on http://azubu.tv . It notifies you when a streamer goes online for your convenience. You can add and remove streamers. 

This is known to work on both linux and Windows. Unknown on Mac.

The application alone uses little CPU and RAM. However, the more streams you add, the more it increases. The application checks every minute to see if a streamer live or not.
